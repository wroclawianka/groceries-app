   - :wrench: React, Redux, Node.js, MongoDB

## Getting started

### Preparation

 Main Directory 
```
npm install
```
Frontend
```
cd frontend && npm install
```
Backend
```
cd backend && npm install
```


## Start
Main directory (backend and frontend will run with concurrently)
```
npm start
```
